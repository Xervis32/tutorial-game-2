# My Second Unity 2D Game
A simple 2D space shooter made in Unity, based on a YouTube [tutorial](https://www.youtube.com/watch?v=P377cNMG4lg&t=19706s).
Created to learn core mechanics and prepare for recreating Warblade in a future project.
